# 👨‍🦲 CarecaAI

O **CarecaAI** é um tutor de programação inteligente, focado em ajudar desenvolvedores (de qualquer nível) a resolver bugs, entender lógica complexa e aprender novas tecnologias com uma abordagem informal e direta.

---

## 🚀 Funcionalidades da Versão 0.2

* **Arena de Bugs:** Desafia os teus conhecimentos num modo interativo onde a IA gera bugs propositados para tu resolveres.
* **Gestão de Conversas:** Cria, renomeia e apaga sessões de chat. O teu histórico é persistido localmente no navegador.
* **Motor Inteligente:** Baseado no `Qwen/Qwen2.5-Coder-32B-Instruct`, capaz de lidar com qualquer linguagem de programação ou infraestrutura.
* **Interface Premium:** Design inspirado nas melhores ferramentas de chat do mercado, com realce de sintaxe profissional e suporte a mobile.

---

## 🛠️ Tecnologias Utilizadas

- **Front-end:** [Next.js 16](https://nextjs.org/), [Tailwind CSS v4](https://tailwindcss.com/)
- **IA/Back-end:** [Hugging Face Inference API](https://huggingface.co/)
- **Mobile:** [Capacitor](https://capacitorjs.com/) para compilação nativa Android.
- **Persistência:** `localStorage` (Browser API).
